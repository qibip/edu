1.  Углубленное изучение JavaScript
    1.1. Основы (ES6+)

        let/const, стрелочные функции, деструктуризация, шаблонные строки.
        ⏳ 3 дня (6 часов).

    1.2. Работа с массивами и объектами

        Методы массивов (map, filter, reduce, и т.д.), обход объектов (for...in, for...of).
        ⏳ 4 дня (8 часов).

    1.3. Асинхронное программирование

        Промисы, async/await, работа с fetch, обработка ошибок. 
        ⏳ 5 дней (10 часов).

    1.4. Продвинутые концепции

        Замыкания, контекст this, прототипы, классы. 
        ⏳ 5 дней (10 часов).

    1.5. Работа с DOM

        Создание, изменение, удаление элементов. События, обработчики.
        ⏳ 4 дня (8 часов).

2.  Практика через задачи
    2.1. Алгоритмы

        Реализация сортировок, поиска, работы с числами и строками.
        ⏳ 6 дней (12 часов).

    2.2. Интерактивные приложения

        Мини-проекты: калькулятор, To-Do List.
        ⏳ 5 дней (10 часов).

    2.3. Работа с API

        Пример: получение данных с публичного API (погода, фильмы, новости).
        ⏳ 4 дня (8 часов).

    3.  Переход к TypeScript
        3.1. Добавление типов в JS-код

        Типы данных, аннотация функций, массивы, объекты.
        ⏳ 4 дня (8 часов).

    3.2. Работа с интерфейсами и дженериками

        Интерфейсы, объединение типов, дженерики.
        ⏳ 5 дней (10 часов).

    3.3. Практика на TS

        Переписывание проектов с JS на TS, работа с библиотеками.
        ⏳ 5 дней (10 часов).

Итоговое время

    Теория и практика JavaScript: ~31 день (62 часа).
    Переход к TypeScript: ~14 дней (28 часов).
    Общее время: ~45 дней (90 часов).
